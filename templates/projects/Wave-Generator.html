{% extends "project_details.html" %} {% block project_content %} Wave-Generator is a simple command line sine wave generator written in rust using the crossterm library. Thanks to the crossterm nature of this project, Wave-Generator works perfectly in Windows, MacOS, Linux, and BSD.

Running

To run Wave-Generator you'll have to have a working version of cargo installed on your system. After that, you can simply cd inyou the project's directory and run cargo run
